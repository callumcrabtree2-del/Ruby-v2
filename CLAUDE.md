# Ruby v2 — Project Instructions

## Stack
- Frontend: Next.js (TypeScript)
- Backend: FastAPI (Python)
- Styling: Tailwind CSS

## Project context
Ruby is a personal AI assistant with a deep space theme. UI uses custom CSS animations.
Features: document reading, image support, voice input, persistent memory, chat export, Code Laboratory.

## UI/UX Skill
This project uses the ui-ux-pro-max skill defined in ~/.claude/CLAUDE.md.

For any UI task, always run the design system search first:
```bash
python3 ~/.claude/skills/ui-ux-pro-max/scripts/search.py "AI assistant chat interface dark space" --design-system -p "Ruby v2"
```

## Design notes
- Deep space theme — dark backgrounds, cosmic colour palette
- Custom CSS animations throughout
- Prioritise: dark mode contrast, animation performance, accessibility labels
